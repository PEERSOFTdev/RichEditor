# User-Requested Features

This directory tracks features requested by users outside the main phase roadmap.
Each file is one request. Agents pick up the plan, implement it, and mark it done.

## Naming

`UR-NNN_short_description.md` — three-digit serial, lower-snake description.

## Status values

| Status | Meaning |
|--------|---------|
| Planned | Approved, not yet started |
| In Progress | Implementation underway |
| Done | Merged, tested |
| Rejected | Won't fix, with reason |
