# Development Phases

This project is organized into phases. The list below reflects completed work and planned work based on recent commits and the notes in `docs/notes/`.

## Completed Phases

- **Phase 1** - Core editor foundation (Win32/RichEdit, file I/O, shortcuts, status bar).
- **Phase 2** - External filter system core.
- **Phase 2.5** - Interactive REPL mode.
- **Phase 2.6** - Session resume (automatic recovery on shutdown).
- **Phase 2.7** - Template system (templates, picker, File->New integration).
- **Phase 2.8** - Configurable RichEdit library selection.
- **Phase 2.8.1** - RichEdit version detection.
- **Phase 2.8.5** - About dialog enhancements + RichEditD2DPT support.
- **Phase 2.9.1** - Find dialog with escape sequences and history.
- **Phase 2.9.2** - Replace functionality (Replace/Replace All + undo), with performance fixes.
- **Phase 2.9.3** - Bookmarks (toggle/navigate/clear) with per-file persistence.
- **Phase 2.9.4** - Go to Line dialog (Ctrl+G).
- **Phase 2.10** - Configurable date/time formatting (DateFormat/TimeFormat/DateTimeTemplate).

## Planned / Open Phases

- **Phase 2.9.x** - Any follow-up items called out in `docs/notes/PHASE_2.9_PLAN.md`.
- **Phase 2.11** - Wrap ruler (twips/column-based wrap) plan in `docs/notes/PHASE_2.11_WRAP_RULER_PLAN.md`.
