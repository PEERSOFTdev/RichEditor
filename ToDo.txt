3. Configurable Date/Time Format: Let the Insert Date/Time feature be configurable via INI file parameters specifying separate strings for date, time, and template for them (like "%date%, %time%").
   - Documentation:
     - To control the %DATE%: https://learn.microsoft.com/en-us/windows/win32/api/datetimeapi/nf-datetimeapi-getdateformatex
       - Variables %LONGDATE%, %SHORTDATE%, %YEARMONTH% and %MONTHDAY% can be used in INI file template and will correspond to individual dwFlags.
       - Format String Description: https://learn.microsoft.com/en-us/windows/win32/intl/day--month--year--and-era-format-pictures
     - To control the %TIME%: https://learn.microsoft.com/en-us/windows/win32/api/datetimeapi/nf-datetimeapi-gettimeformatex
3. Support Tab Completion For Active REPL Filter: Triggered by pressing TAB while on the Prompt Line after the detected prompt; sends TAB to the REPL Filter and captures its stdout, inserting it into the current line at the cursor position
