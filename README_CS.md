# RichEditor

RichEditor je lehký, plně přístupný textový editor pro Windows, postavený na komponentě Win32 RichEdit. Pracuje s prostým textem v kódování UTF-8.

## Uživatelská příručka

Podrobná příručka v češtině: `docs/USER_MANUAL_CS.md`

## Poznámka k jazykům

Aplikace RichEditor obsahuje české i anglické rozhraní; jazyk se volí automaticky podle nastavení Windows. Uživatelská příručka je k dispozici v obou jazycích. Ostatní dokumenty (`LICENSE`, `CHANGELOG.md`, `README.md`) jsou pouze v angličtině.
